# eCommerce Laptop Shop
E-Commerce Laptop Shop is online shop for sell on laptops.

## View the application
Link: https://laptop-shop.netlify.app/
- version: demo

## Dependencies
`Server / Backend`
- bcrypt
- dotenv
- express
- jsonwebtoken
- mongoose

`Client / Frontend`
- material-ui
- react-router-dom
- react-responsive-carousel
- axios
